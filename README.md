To run locally:
- npm run dev
- make sure mongo is set up and running

This app was created with:
- create-react-app
- React
- Fetch (aka XHR)
- SASS
- Webpack
- Node/Express
- MongoDB / Mongoose SDK

Backend
- /server.js (Node)
